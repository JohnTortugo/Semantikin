Detailed Description

Base class for any class that needs to traverse the AST. This class define an interface following the visitor pattern.

virtual void AstNodeVisitor::visit ( Parser::CompilationUnit *  module)
pure virtual

This method start the traversing of the whole AST tree. It calls the visit/accept method for the components of the module, i.e., the functions.
